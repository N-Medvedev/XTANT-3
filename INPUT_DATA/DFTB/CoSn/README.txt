The provided SK files were optimized to reproduce the electronic band structures of hcp Co, alpha-Sn, beta-Sn, and CoSn.
Please note that the repulsive potential was not optimized in this parameter set.
Therefore, when using these files, ensure that "PolynomialRepulsive = SetForAll{Yes}" is specified in the Hamiltonian block of your dftb_in.hsd file.
These files should not be used for structural optimization or molecular dynamics simulations.
